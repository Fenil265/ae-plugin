== Description ==
Official plugin of the largest event discovery platform "All Events in City".

1) Events in City - Engage your visitors by displaying the best events from All Events in City right into your site. You can select the height and width below. If your site is of specialized nature, you can also select the category. It is as easy as it can get.

You can display events by category like Sports, Music, Concerts, Parties, Theatre, Performance, Art, Exhibitions, Business, Workshops, Festivals, Meetups events.

Now onwords you can embed your events onto your website using simple shortcode.

2) Events by Organizer - Plugin lets you show your upcoming events.You can select the height and width below.It is as easy as it can get... and free!

Also Organizer can display their upcoming and past events by adjusting height and width. 

Allevents.in has the largest pool of events. Covers 30million events from 30,000 cities worldwide.

<h3>Shortcodes</h3>

1) Events in City
<blockquote><p>[city-events city='New York' category='All' width='100%' height='500px']</p></blockquote>  

<blockquote><p>[embed-events event_id='192895701069404' width='100%' height='500px']</p></blockquote>  

2) Events by Orgnizer
<blockquote><p>[org-events organizer_id='198763' width='100%' height='500px']</p></blockquote>

The plugin allows you to select city and category of events. You can adjust/set height and width of the box from widget section.

Website: http://allevents.in/

<strong><p>Note* :</strong> For other frameworks or custom websites, you can download the iframe plugin from <a href="http://allevents.in/pages/organizer-plugin" target="_blank">All Events in City | Organizer Plugin</a></p>

== Installation ==

1. Unpack events-in-city.zip and upload its content to the `/wp-content/plugins/` directory
2. Activate the plugins called 'AE - Events In City' , 'AE - Events by Organizer' and 'AE - Shortcode Generator' through the 'Plugins' menu in WordPress
3. You can see 'Events In City' and 'Events by Organizer' widgets in appearance -> widgets -> Events In City/Events by Organizer.
4. Drag and drop the widget, wherever you want to display events.
